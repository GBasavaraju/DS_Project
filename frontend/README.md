# UI for DS project

## Setup
From folder 'frontend', run
```bash
npm install
npm start
```

## Libraries used

Styling: 
1. https://tailwindcss.com/ - for general styling  
2. https://daisyui.com/ - for components

Icons:  
https://heroicons.com/ - Just copy SVG and paste into HTML

Chart plotting:  
https://www.chartjs.org/ - Tentative
